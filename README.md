  MultiCheckin
===========================================
  Курсовой проект по Android-у. 
  Основная цель проекта - сделать возможность постить фотографии (уже ) с комментариями с телефона сразу в нескольно социельных сетей одновременно.
  
#### Поддерживаемые социальные сети:
* ВКонтакте
* Facebook
* Одноклассники
* Twitter
